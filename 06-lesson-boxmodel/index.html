06 Lesson Box Model

El contenido de un documento HTML se encuentra dividido en cajas de la siguiente forma:

  • Contenido, propiamente dicho
  • Padding: espacio entre el contenido y el border.
  • Border: espacio entre el padding y el margin.
  • Margin: espacio entre el border y el margin de otra caja de contenido

También podemos asignar valores a partes concretas del padding, margin y border.

Podemos usar algo como "margin-right:" o "padding-top:" pero también podemos utilizar "margin: 10px 50px 5px 6px;" para asignar valores concretos a cada una de las partes del margin, ahorrándonos tener que escribirlo uno a uno.
